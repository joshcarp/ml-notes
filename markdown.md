# Markdown Examples

This page demonstrates various Markdown features with our modern theme.

## Text Formatting

You can write text in **bold** or *italic*. You can also use ***bold and italic***.

## Lists

Unordered list:
* Item 1
* Item 2
  * Subitem 2.1
  * Subitem 2.2
* Item 3

Ordered list:
1. First item
2. Second item
   1. Subitem 2.1
   2. Subitem 2.2
3. Third item

## Code

Inline code: `print("Hello, World!")`

Code block:
```python
def greet(name):
    """Greet someone"""
    return f"Hello, {name}!"

print(greet("Jupyter User"))
```
